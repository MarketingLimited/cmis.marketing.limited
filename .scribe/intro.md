    # Welcome to CMIS Marketing API

    This API provides a unified interface for managing marketing campaigns across multiple platforms including Google Ads, Meta (Facebook/Instagram), TikTok, LinkedIn, Twitter, and Snapchat.

    ## Key Features

    - **Multi-Platform Integration**: Connect and manage campaigns across 6 major advertising platforms
    - **Auto-Sync System**: Automatic data synchronization every hour for metrics, every 4 hours for campaigns
    - **Unified Dashboard**: Single endpoint to view all organization metrics and performance
    - **Event-Driven Architecture**: Real-time updates and automatic cache invalidation
    - **Content Scheduling**: Schedule and publish content across social media platforms
    - **Unified Campaign API**: Create complex campaigns with ads and content in a single API call

    ## Authentication

    All API requests require Bearer token authentication via Laravel Sanctum.
