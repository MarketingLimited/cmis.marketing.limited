## نظرة عامة على نظام Instagram داخل CMIS

يهدف هذا النظام إلى ربط منصة CMIS مباشرة مع واجهة Instagram Graph API بحيث يمكن جلب البيانات التحليلية، المنشورات، والإحصائيات بدقة عالية عبر أوامر Artisan. يعتمد النظام على مكونين رئيسيين:

1. **InstagramService**: المسؤولة عن الاتصال بـ Graph API وجلب البيانات.
2. **InstagramApiCommand**: الأمر المخصص داخل Laravel Artisan الذي يتفاعل مع الخدمة ويعرض النتائج.

يدعم النظام جميع أنواع الوسائط (Reel، صورة، فيديو، كاروسيل)، ويستطيع تحديد الفترات الزمنية، وفرز المنشورات حسب الإحصائيات المختلفة مثل Reach أو Likes.

### ⚙️ مكونات النظام
- **cmis.integrations**: الجدول الذي يحتوي على بيانات الاتصال الخاصة بحسابات Instagram.
- **InstagramService**: تتولى الجلب، المعالجة، والإحصاءات.
- **InstagramApiCommand**: الأمر التنفيذي في Laravel.
- **Logs**: لتتبع الأخطاء ومراحل التنفيذ.

يتم الوصول إلى كل هذه العمليات من خلال أوامر Artisan دون الحاجة لأي واجهة رسومية.
